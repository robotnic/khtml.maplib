khtml.maplib.overlay.WMS(options)
WMS Overlay. Overlay the map with a layer from a wms server. Only WGS84 (EPSG:4326) is supported. WMS URL's are required to include '&SRS=EPGSG:4326'. WMS Specification: http://www.opengeospatial.org/standards/wms Example: 
	var wms = new khtml.maplib.overlay.WMS(
		{
			opacity: 0.6, 
			url: 'http://wms.geo.admin.ch/',
			parameters: {'FORMAT':'image/png', 'STYLES': 'default'},
			layers: ['xyz','abc']
		}
	);
	map.addOverlay(wms);
Parameters:
{Object} options
Properties: 'url' => url to wms server, use as . The following parameters are appended to the url: '&BBOX=...&WIDTH=...&HEIGHT=...&xyz=...'. 'opacity' => transparency of image where 0 is fully transparent and 1 is not transparent. 'class' => class is used for the image tag.
