Class Detail
khtml.maplib.overlay.InfoWindow(InfoWindowOptions)
Generates an InfoWindow-object with the specified InfoWindowOptions.
Can be opened on any anchorObject, that exposes position and pixelOffset function. 
Minimum code to open an infowindow on a marker:
   

  var infowindow = new mr.overlay.InfoWindow({
  	content: 'Text in the InfoWindow'
  });
  
  marker.attachEvent( 'click', function() {
  	// Opening the InfoWindow
  	infowindow.open(map, this);
  });
Parameters:
{khtml.maplib.overlay.InfoWindowOptions} InfoWindowOptions
Options for InfoWindow
See:
Example (click on the marker): Infowindow}
Method Detail
clear()
Remove the infowindow from the map. Infowindow still exists, will still be rendered.
close()
Closes the infowindow. Will not be rendered any more.
open(mapObj, anchorObject)
Open the infowindow on the map at an anchorObject. The infowindow is moved with the anchorObject. The anchorObject is not necessarily needed. The infowindow can also be positioned via the position property in the InfoWindowOptions.
Parameters:
{khtml.maplib.Map} mapObj
The map to open the infowindow in.
{Object} anchorObject Optional
A object of the map that provides a getPosition()-method.
render()
Render infowindow, pointer and shadow. Automatically called by the map.
setContent(content)
Set the content for the infowindow.
Parameters:
{String} content
Any string, may also contain HTML-tags.
