khtml.maplib.util.Urlparam(map)
URL Parameter Helper. understands following GET-parameters:
lat: Latitude for map-center
lon: Longitude for map-center
zoom: Zoom-level
mlat: Latitude to place marker
mlon: Longitude to place marker
 
map with center and zoomlevel via urlparam:
 .../map.html?lat=48&lon=14&zoom=9
 
map with center/zoomlevel and marker via urlparam:
 .../map.html?lat=48&lon=14&zoom=9&mlat=48&mlon=14
Parameters:
{khtml.maplib.map} map
to apply parameters on
See:
Example with Lat/Lng and Zoom via Urlparam: urlparam.html?lat=48&lon=14&zoom=9
Example with Marker: urlparam.html?lat=48&lon=14&zoom=9&mlat=48&mlon=14
